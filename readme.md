# cheesy-metal

> Cheesy metal band name generator

## Install

```
$ npm install --save cheesy-metal
```


## Usage

```js
var cheese = require('cheesy-metal');

cheese();
//=> 'Munster Toad'
```


## API

### ()

Type: `string`

Random cheesy metal band name.


## CLI

```
$ npm install --global cheesy-metal
```

```
$ cheesy-metal --help

  Examples
    $ metal-name
    Corrosive Friesian
```
